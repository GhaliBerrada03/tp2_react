# 🚀 TP React - Composants et Hooks

## 📌 Description

Ce projet React contient plusieurs exercices pour pratiquer :

* les composants
* les props
* les événements
* les hooks (`useState`, `useEffect`)

---

## 🛠️ Installation

```bash
npm install
npm start
```

Puis ouvrir : http://localhost:3000

---

## 📂 Structure

```bash
src/
│── App.js
│── AffichageDynamique.js
│── Compteur.js
│── CompteurEffet.js
│── FormulaireInscription.js
│── FormulaireNom.js
│── MessageBouton.js
```

---

## 📸 Fonctionnalités

* 🔁 Affichage dynamique au clic
* 📝 Formulaire contrôlé (prénom + email)
* 🔢 Compteur de clics
* ⚡ useEffect pour modifier le titre
* 🎯 Gestion des événements (boutons)

---

## 🎯 Objectif

Apprendre les bases de React :

* créer des composants
* gérer l’état avec `useState`
* utiliser `useEffect`
* manipuler les événements
